About This Website

This is a static website built using only HTML and CSS. No backend, no database, no JavaScript.

Why Static Website?

  • Fast and lightweight
  • Easy to host
  • Perfect for learning Docker, IIS, Nginx
  • Secure (no server-side code)

Deployment Options

You can deploy this website using:

  • Apache / HTTPD
  • Nginx
  • IIS (Windows Server)
  • Docker container
